                           USAGE INSTRUCTIONS
=================================================================================

1. ACCESSING THE APPLICATION:
   - Navigate to home page: http://localhost:5000
   - You'll see the login page with demo credentials
   - Use provided credentials to login

2. NAVIGATION:
   - Use sidebar for main navigation
   - Different menus based on role (Admin, HR, Employee)
   - Topbar shows user profile and notifications

3. COMMON OPERATIONS:

   - Attendance: Check-in/out, view records
   - Leaves: Request, approve, reject
   - Payroll: Generate, view, export
   - Reports: Generate custom reports, export data

4. ROLE-BASED ACCESS:
   - Admin: Full system access
   - HR: Employee and payroll management
   - Employee: Personal information only
